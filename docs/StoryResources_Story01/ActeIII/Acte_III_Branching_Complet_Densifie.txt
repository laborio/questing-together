ACTE III — CE QUI EST ENTERRÉ ENSEMBLE
BRANCHING COMPLET + DENSIFICATION (STYLE AVENTURE / JDR)

============================================================
PRINCIPE GÉNÉRAL DE L’ACTE III
============================================================

L’Acte III ne garantit PAS une confrontation.
Il garantit une CONVERGENCE.

Selon les actes précédents, le groupe peut :
- découvrir le site final volontairement
- y être conduit par erreur
- ne jamais comprendre ce qu’il représente
- arriver trop tard
- devenir allié du processus
- devenir son principal obstacle

============================================================
SCÈNE 1 — LA CONVERGENCE DES PISTES
============================================================

Les indices accumulés (routes, villages, pilleurs, reliques) pointent tous vers une même zone :
- terres pauvres
- anciennes carrières
- champs abandonnés
- zones sans population stable

CHOIX :
A) Suivre les chariots  
B) Interroger un intermédiaire  
C) Suivre un membre du culte  
D) Ignorer et gérer une crise locale  

CONSÉQUENCES :
A → découverte directe du site  
B → informations partielles, retard  
C → découverte indirecte, embuscade possible  
D → le rituel progresse sans le groupe

============================================================
SCÈNE 2 — LE LIEU SANS NOM
============================================================

Ancienne carrière / ravin sec.
Aucun symbole religieux explicite.
Organisation discrète.

ÉTATS POSSIBLES À L’ARRIVÉE :
- site calme (avant le rituel)
- site en activité (préparatifs)
- site presque achevé
- site désert (trop tard ou abandonné)

INDICES :
- Clous de Clôture en place
- Sceaux Funéraires rares
- Ballots alignés
- Tissus noirs en repères

============================================================
SCÈNE 3 — RENCONTRES POSSIBLES
============================================================

Selon le timing et la discrétion :

A) Ouvriers ordinaires  
B) Gardiens du site  
C) Membres du culte (non fanatiques)  
D) Le Custodien  
E) Aucun (site abandonné)

============================================================
SCÈNE 4 — LE CUSTODIEN (OPTIONNEL)
============================================================

Le Custodien n’est rencontré QUE si :
- le groupe arrive à temps
- le groupe n’a pas semé une panique totale
- le site est encore actif

DIALOGUE CLÉ :
« Ce n’est pas un vol.
Ce n’est pas un culte.
C’est une réparation. »

CHOIX :
A) Attaquer immédiatement  
B) Arrêter / capturer  
C) Discuter / négocier  
D) Observer en silence  
E) Feindre l’alliance  

============================================================
SCÈNE 5 — LE RITUEL FINAL (OU SON ABSENCE)
============================================================

Le rituel n’est PAS spectaculaire.
Il s’agit :
- d’alignements
- d’enterrements coordonnés
- d’activation passive des reliques

ÉTATS :
- Rituel interrompu
- Rituel modifié
- Rituel achevé
- Rituel corrompu

============================================================
SCÈNE 6 — CONFLIT FINAL
============================================================

Peut être :
- combat direct
- chaos contrôlé
- sabotage discret
- fuite sous pression

CHOIX :
A) Détruire toutes les reliques  
B) Détruire certaines reliques  
C) Déplacer les reliques  
D) Protéger le rituel  
E) Quitter les lieux  

============================================================
SCÈNE 7 — ÉTATS DE SORTIE IMMÉDIATS
============================================================

- culte dissous
- culte transformé
- culte renforcé
- phénomène déplacé
- groupe célébré
- groupe pourchassé
